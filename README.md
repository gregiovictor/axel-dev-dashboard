# Axel — Assistente Virtual

Axel é um dashboard de produtividade que centraliza **projetos, tarefas e ferramentas de desenvolvimento** em um único lugar.

O objetivo é evitar o dispersamento de informações entre várias plataformas e facilitar o acompanhamento do progresso dos projetos.

## Funcionalidades

* Criar e gerenciar projetos
* Lista de tarefas por projeto
* Barra de progresso baseada nas tarefas concluídas
* Menu lateral com atalhos para ferramentas
* Saudação automática baseada no horário
* Persistência de dados no navegador

## Tecnologias

* JavaScript
* LocalStorage
